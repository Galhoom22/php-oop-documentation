## Interview Question

What are PHP attributes, and what problem do they solve?

## What The Interviewer Is Testing

Whether you know how PHP attaches structured metadata to code.

## A Strong Answer Should Cover

- Attributes are structured metadata written above classes, methods, or other code elements.
- They do not directly replace normal logic.
- They are useful when frameworks or tools need to read extra instructions from code.

## Practice Task

Explain the difference between an attribute and a normal method call.
