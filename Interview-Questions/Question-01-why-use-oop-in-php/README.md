## Interview Question

Why use OOP in PHP instead of only functions?

## What The Interviewer Is Testing

The practical problem OOP solves in growing PHP codebases.

## A Strong Answer Should Cover

- Data and behavior can be grouped together in one unit.
- OOP reduces scattered procedural logic when the project grows.
- OOP is useful when the code has repeated concepts such as users, posts, orders, or invoices.

## Practice Task

Explain a small procedural PHP example that becomes hard to maintain, then describe how a class would organize it.
