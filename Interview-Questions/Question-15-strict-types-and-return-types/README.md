## Interview Question

Why are type declarations and return types important in OOP PHP?

## What The Interviewer Is Testing

Whether you use types to make object contracts clear and safer.

## A Strong Answer Should Cover

- Parameter types say what kind of value or object a method accepts.
- Return types say what the method promises to return.
- `strict_types=1` helps catch wrong scalar values instead of silently converting them.
