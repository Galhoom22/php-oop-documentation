## Interview Question

When would you use an anonymous class, and how does object iteration work?

## What The Interviewer Is Testing

Whether you know one-time object definitions and visible property iteration.

## A Strong Answer Should Cover

- Anonymous classes are useful for small one-off implementations.
- They avoid naming a class that has no reusable meaning.
- `foreach` over an object iterates visible properties unless custom iteration is implemented.

## Practice Task

Describe one testing or callback scenario where an anonymous class could be useful.
