## Interview Question

What is the difference between a class and an object?

## What The Interviewer Is Testing

Whether you understand the blueprint-versus-real-instance relationship.

## A Strong Answer Should Cover

- A class defines structure and behavior.
- An object is a specific instance created from that class.
- Many objects can be created from the same class and hold different data.

## Practice Task

Write a tiny class name and describe two different objects that could be created from it.
