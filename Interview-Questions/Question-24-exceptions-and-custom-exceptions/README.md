## Interview Question

How do exceptions work in OOP PHP, and when should you create a custom exception?

## What The Interviewer Is Testing

Whether you can signal and handle errors clearly.

## A Strong Answer Should Cover

- `throw` creates an exceptional failure signal.
- `try/catch/finally` separates normal flow from error handling.
- Custom exceptions make specific failure types easier to catch and understand.
