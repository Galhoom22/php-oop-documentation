What is encapsulation, and how does it protect data?

## What The Interviewer Is Testing

## A Strong Answer Should Cover

- Encapsulation hides internal details behind safe public methods.
- Setters or methods can validate data before changing object state.
- Public properties can allow invalid data when used carelessly.
