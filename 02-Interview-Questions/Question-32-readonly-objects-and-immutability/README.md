## Interview Question

What are readonly classes, and why is immutability useful?

## What The Interviewer Is Testing

Whether you understand object state that cannot be changed after construction.

## A Strong Answer Should Cover

- Readonly properties can only be assigned once.
- Readonly classes make all instance properties readonly.
- Immutable objects reduce accidental state changes and can be easier to reason about.

## Practice Task

Explain why a `Money` value object is a strong candidate for readonly design.
