## Interview Question

What is the concept of Abstraction?

## What The Interviewer Is Testing

Whether you can explain abstraction as hiding details and exposing only what matters.

## A Strong Answer Should Cover

- Abstraction means focusing on the essential idea and hiding unnecessary internal details.
- In PHP OOP, abstract classes and interfaces are common tools for defining abstract contracts.
- Callers work with the public idea or contract, not with every internal implementation detail.

## Practice Task

Explain abstraction using a payment example: the caller knows `pay()`, not the bank details inside.
