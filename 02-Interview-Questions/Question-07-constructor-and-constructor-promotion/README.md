## Interview Question

What is a constructor, and why is constructor promotion useful?

## What The Interviewer Is Testing

Whether you know how objects receive required data at creation time.

## A Strong Answer Should Cover

- `__construct()` runs automatically when the object is created.
- Constructor parameters force required data to be provided early.
- Constructor promotion reduces repeated property declaration and assignment code.

## Practice Task

Write a small class with a promoted constructor and explain each promoted property.
