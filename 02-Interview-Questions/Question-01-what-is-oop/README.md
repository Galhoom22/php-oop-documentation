## Interview Question

What is OOP?

## What The Interviewer Is Testing

Whether you can define object-oriented programming clearly without memorized buzzwords.

## A Strong Answer Should Cover

- OOP is a programming style that models code as objects that combine data and behavior.
- A class is the blueprint; an object is a living instance created from that blueprint.
- OOP helps organize growing code around real concepts such as users, orders, or invoices.
