## Interview Question

What is the difference between an abstract class and an interface?

## What The Interviewer Is Testing

Whether you can choose the right kind of contract.

## A Strong Answer Should Cover

- An abstract class can contain shared implementation and abstract methods.
- An interface describes required methods without storing normal implementation details.
- A class can extend one parent class but can implement multiple interfaces.

## Practice Task

Explain whether `PaymentGateway` should be an interface, abstract class, or normal class.
