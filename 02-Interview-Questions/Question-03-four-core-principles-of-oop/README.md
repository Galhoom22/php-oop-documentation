## Interview Question

What are the four core principles of OOP?

## What The Interviewer Is Testing

Whether you know the classic OOP pillars and can name them with a short, correct meaning for each.

## A Strong Answer Should Cover

- Encapsulation: keep internal data hidden and expose a safe public surface.
- Inheritance: build a new class from an existing one and reuse shared structure or behavior.
- Polymorphism: treat different objects through a shared type or contract when they can respond differently.
- Abstraction: hide complex details and expose only the essential idea or contract.

## Practice Task

Name the four principles and give one plain-English example for each.
