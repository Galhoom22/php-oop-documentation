## Interview Question

What is the difference between properties and methods?

## What The Interviewer Is Testing

Whether you can separate object data from object behavior.

## A Strong Answer Should Cover

- A property stores data inside the object.
- A method is a function inside the class that performs behavior.
- Methods often read or change properties through `$this`.

## Practice Task

Create a simple object idea and list three properties and two methods it should have.
