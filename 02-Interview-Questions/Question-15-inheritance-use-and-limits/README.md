## Interview Question

What is inheritance, and when can it become a bad choice?

## What The Interviewer Is Testing

Whether you understand `extends` and avoid forcing weak parent-child relationships.

## A Strong Answer Should Cover

- Inheritance creates a child class that receives behavior from a parent class.
- PHP supports single inheritance only.
- Inheritance is best for true is-a relationships, not for sharing random code.

## Practice Task

Decide whether `AdminUser extends User` is reasonable and what risk it might introduce.
