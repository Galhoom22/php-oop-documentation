# PHP OOP Documentation

> This repo is for those who want to study everything related to OOP in PHP with modern **PHP 8.5+** syntax. All lectures follow the [official PHP documentation](https://www.php.net/manual/en/langref.php), with interview questions and a final project to master it all.

---

## Purpose

Object-Oriented Programming is how real PHP applications are built — from Laravel and Symfony to custom APIs and SaaS products. Yet most tutorials either skip modern language features or teach outdated patterns.

**This repository exists to give you one complete, structured path:**

- Learn every core OOP concept in PHP using **modern syntax** (PHP 8.5+)
- Study from material grounded in the **official PHP manual** — not random blog posts
- Practice with **realistic interview questions** so you can explain what you know
- Prove your skills by building a **real-world capstone project** from scratch

No framework. No shortcuts. Just PHP, objects, and the confidence to read and write professional OOP code.

## How to Start

1. Open [`PHP-OOP-Lectures.md`](PHP-OOP-Lectures.md) and begin with **Lecture 1**
2. Work through **one lecture per session** — write the code in the matching `Lectures/` folder
3. After all 31 lectures, move to [`PHP-OOP-Interview-Questions.md`](PHP-OOP-Interview-Questions.md)
4. When interview prep is complete, start the final project in [`PHP-OOP-Final-Project.md`](PHP-OOP-Final-Project.md)

> **Important:** You write all the code. This repo provides the curriculum, structure, and guidance — your practice files live in the workspace folders.
